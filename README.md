# React Blog Site

## Introduction

This project is a personal blog website built using React components and props.

## Features

- Component-based structure
- Dynamic rendering using props
- Reusable Article components
- Blog data stored separately in blog.js

## Component Tree

App
├── Header
├── About
└── ArticleList
    └── Article

## Installation

1. Clone the repository

git clone 

2. Navigate into the project folder

cd react-blog-site

3. Install dependencies

npm install

4. Start the development server

npm run dev

## Testing

Run tests using:

npm test

## Technologies Used

- React
- Vite
- JavaScript
- JSX
